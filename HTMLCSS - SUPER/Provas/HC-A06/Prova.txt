[H/C-06] Crie uma página HTML que mostre como aplicar responsividade e cores em CSS. Para isso, siga as instruções abaixo:

Use media queries para ajustar o tamanho do texto em diferentes larguras de tela (desktop e mobile).
Defina uma paleta de cores usando variáveis CSS e aplique essas cores ao fundo e aos textos da página.
Utilize unidades flexíveis (como vw, vh, em, ou rem) para o tamanho de fonte e os espaçamentos, garantindo que o layout se adapte a diferentes dispositivos.
